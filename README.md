# Simple Interest Calculator

## Description
This repository contains a shell script `simple-interest.sh` to calculate **simple interest** using the formula:

SI = (P × R × T) / 100

where:  
- P = Principal  
- R = Rate of Interest (%)  
- T = Time (years)  

This project demonstrates basic shell scripting and GitHub repository setup.
